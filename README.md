# provaN1
# Prova pratica da matéria de linguagem de programação para internet


Esse código é uma tela de cadastro simples que quando fazemos o login, a tela de cadastro some automaticamente

para o código funcionar precisa instalar ceras requisições:

//copie esse código para dentro do terminal
npm init -y
npm install express cors

//logo depois rode o servidor, tem que aparecer a mensagem "Servidor rodando na porta 3000"
node server.js

após isso, você pode colocar 2 usuários para testar:
